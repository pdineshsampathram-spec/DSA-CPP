# C++ STL Vector Practice 🚀

This repository contains beginner-friendly practice problems to understand and master **C++ STL `vector`**.

The goal of this practice set is to build a strong foundation in:

* `vector`
* `push_back()`
* `size()`
* traversing vectors
* sorting vectors
* basic problem solving with arrays

These concepts are fundamental for **Data Structures & Algorithms (DSA)** and **competitive programming**.

---

# 📌 Problems Implemented

## 1️⃣ Store and Print Vector

**Problem**

Take `n` numbers as input, store them in a vector, and print them.

Example

Input

```
5
1 3 5 7 9
```

Output

```
1 3 5 7 9
```

Concepts Used

* vector
* loops
* input/output

---

## 2️⃣ Sum of Vector Elements

**Problem**

Find the sum of all elements in a vector.

Example

Input

```
4
2 5 7 3
```

Output

```
17
```

Concepts Used

* vector traversal
* accumulation

---

## 3️⃣ Find Maximum Element

**Problem**

Find the largest number in the vector.

Example

Input

```
5
4 8 2 10 6
```

Output

```
10
```

Concepts Used

* comparisons
* vector traversal

---

## 4️⃣ Reverse a Vector

**Problem**

Reverse the elements of a vector.

Example

Input

```
5
1 2 3 4 5
```

Output

```
5 4 3 2 1
```

Concepts Used

* vector indexing
* reversing logic

---

## 5️⃣ Count Even Numbers

**Problem**

Count how many numbers in the vector are even.

Example

Input

```
6
1 4 7 8 10 3
```

Output

```
3
```

Concepts Used

* conditions
* vector traversal

---

## 6️⃣ Remove Duplicate Elements

**Problem**

Remove duplicate elements from the vector.

Example

Input

```
6
1 2 2 3 3 4
```

Output

```
1 2 3 4
```

Concepts Used

* sorting
* removing duplicates

---

## 7️⃣ Sort a Vector

**Problem**

Sort elements of a vector.

Example

Input

```
5
9 2 7 1 5
```

Output

```
1 2 5 7 9
```

Concepts Used

* STL `sort()`
* vector iterators
